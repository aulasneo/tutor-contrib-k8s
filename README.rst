k8s plugin for `Tutor <https://docs.tutor.edly.io>`__
#####################################################

Helper plugin for Kubernetes deployments of Open edX


Installation
************

.. code-block:: bash

    pip install git+https://github.com/aulasneo/tutor-contrib-k8s.git

Usage
*****

.. code-block:: bash

    tutor plugins enable k8s
